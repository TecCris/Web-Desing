Desafio

  1. Incluir ícone DEPOIS do texto dos links que abrem em nova aba
  2. Incluir ícone ANTES do texto dos links que abrem em nova aba
  3. Colorir os textos do labels que veem apos os checkboxes seguindo a seguinte regra
    1. Após o status checked a cor do label deve ser verde
    2. Após o status não checked a cor do label deve ser vermelho
    3. Se o label vem apos um checkbox disable, a cor deve ser azul
    4. Nessa lista ordenada, colocar um background qualquer no terceiro filho(li) desde que esteja contido dentro de outra (ul) ou (ol)

Atenção: Não é permitido alterar nada no HTML (nem mesmo adicionar classes)
